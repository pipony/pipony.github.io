Windows命令行手册

2023-02-14

此笔记为《少数派-生产力超频：Windows 命令行实用手册》l栏目的个人学习笔记，供个人回溯

一些概念

Command Line Interface (CLI) 命令行界面 Console App & Terminal App 控制台应用：没有图形界面，需要在命令提示符中使用。直接双击会闪退。

包管理器： Windows 平台三大包管理器（Chocolatey、Scoop、winget）。它们可以帮助你批量安装、卸载、升级 PC 中的应用，无需再去官网获得安装包。

内核（kernel）：直接和硬件打交道； 壳（shell）：运行命令的东西；可以将cd等命令翻译成计算机能听懂的语言

Windows 上的 cmd、PowerShell 是 shell 的一种。此外，Linux 和 macOS 上有各种各样以 sh 结尾的应用，例如 bash、zsh、fish 以及 csh，其中的 sh 就是 shell 的缩写。 壳的窗口：Windows 控制台（conhost.exe），也称为终端仿真器（terminal emulator），简称终端。

外部命令和shell命令：外部命令不依赖shell存在，shell命令是shell自带的命令
