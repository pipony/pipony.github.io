lpad/rpad函数

此为Oracle函数。l/r(left/right)+pad(填充)，表示从左边/右边填充。

格式：lpad( string, padded_length, [ pad_string ] ) （rpad同理）

参数一：被填充/截取的原字符串

参数二：最终长度

参数三：填充的字符，若无赋值，则默认填充空格

注意：若原字符串长度大于参数二的长度，则lpad从左开始截取指定长度（rpad从右边）
