认识Jar包

概念

Jar：Java Archive File

基于zip文件格式将许多文件合成一个压缩文件.jar，区别是比zip多了一个包含了一个 META-INF/MANIFEST.MF 文件，这个文件是在生成 JAR 文件的时候自动创建的。

特点：

  • 平台无关
  • 加快下载速度
  • 安全：可对jar文件数字签名

分类：

  • 可执行jar包（MANIFEST.MF文件里有主类入口配置：Main-Class: com.example.MyClass）
  • 不可执行jar包
