概念

字符集 vs. 编码方式：编码方式就是将字符集里的字符映射成可以被计算机识别的若干位二进制来表示。

ASCII

American Standard Code for Information Interchange，美国标准信息交换代码

主要用来表示英文或其他西欧语言

表示

  • 使用7位表示128种字符：标准（基础）ASCII 码
  • 使用8位表示256种字符：扩展ASCII 码，多的一位表示附加的128 个特殊符号字符、外来语字母和图形符号

GB2312

为了支持中文的表示，国人发明的

Unicode

为了解决使用不同国家的编码格式所造成的乱码问题，国际编码格式Unicode应运而生。

表示

一个字符用2个字节表示

UTF-8

在表示纯英文时，Unicode比ASCII码多了一倍，很不划算。于是有了变长表示的UTF-8。

UTF-8编码将Unicode字符按数字大小编码为1-6个字节，英文字母被编码成一个字节，常用汉字被编码成三个字节，如果你编译的文本是纯英文的，那么用UTF-8就会非常节省空间。
