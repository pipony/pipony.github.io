背景：我的阿里云linux服务器（CentOS 7.8 64位）之前在宝塔已懒人安装了mysql，现需要在服务器上学习Oracle，所以先记录安装过程（静默安装）。

正式安装之前，可以先检测下是否已有安装Oracle程序。若已有安装oracle，应能看到oracle进程，且一般都会有oracle用户。我在本机上验证如下，是没有安装过的，

6[cindy@iZbp15qc4wmx335c268l5mZ ~]$ yum list installed | grep oracle  # 查看以yum安装的软件列表

创建用户/组

首先看下已有的用户和组，以免出现重复。没有与要创建的用户/组重名的，so放心安装。

1cat /etc/passwd  # 查看所有用户
2cat /etc/group  # 查看所有组

创建用户组oinstall,dba，创建用户oracle

 1[root@iZbp15qc4wmx335c268l5mZ ~]# groupadd -g 200 oinstall  # 组id设为200
 2[root@iZbp15qc4wmx335c268l5mZ ~]# groupadd -g 201 dba
 3[root@iZbp15qc4wmx335c268l5mZ ~]# useradd -u 440 -g oinstall -G dba oracle  # 创建oracle用户，指定uid为400
 4[root@iZbp15qc4wmx335c268l5mZ ~]# passwd oracle  # 设置oracle密码

oracle安装

在此之前，还需要修改下/app/oracle/database/response/db_install.rsp`文件。修改内容参考 https://blog.csdn.net/csd_nuser/article/details/110089960 和 http://www.javashuo.com/article/p-sbcnrnqx-gr.html 。

 1vim db_install.rsp # 以下为可参考的【http://www.javashuo.com/article/p-sbcnrnqx-gr.html】的修改内容，修改内容以具体情况和需求为准
 2oracle.install.responseFileVersion=/oracle/install/rspfmt_dbinstall_response_schema_v11_2_0
 3oracle.install.option=INSTALL_DB_SWONLY
 4ORACLE_HOSTNAME=能够写本机地址
 5UNIX_GROUP_NAME=oinstall
 6INVENTORY_LOCATION=/app/oracle/oraInventory
 7SELECTED_LANGUAGES=zh_CN,en
 8ORACLE_HOME=/app/oracle/product/11.2.0/db_1
 9ORACLE_BASE=/app/oracle
10oracle.install.db.InstallEdition=EE
11oracle.install.db.isCustomInstall=false
12oracle.install.db.DBA_GROUP=dba
13oracle.install.db.OPER_GROUP=oinstall
14oracle.install.db.config.starterdb.type=GENERAL_PURPOSE
15oracle.install.db.config.starterdb.globalDBName=orcl
16oracle.install.db.config.starterdb.SID=orcl
17oracle.install.db.config.starterdb.characterSet=AL32UTF8
18oracle.install.db.config.starterdb.memoryOption=true
19oracle.install.db.config.starterdb.memoryLimit=512
20oracle.install.db.config.starterdb.installExampleSchemas=false
21oracle.install.db.config.starterdb.enableSecuritySettings=true
22oracle.install.db.config.starterdb.password.ALL=oracle2018
23oracle.install.db.config.starterdb.control=DB_CONTROL
24oracle.install.db.config.starterdb.dbcontrol.enableEmailNotification=false
25oracle.install.db.config.starterdb.automatedBackup.enable=false
26DECLINE_SECURITY_UPDATES=true

1[root@iZbp15qc4wmx335c268l5mZ response]# vim /app/oracle/database/response/db_install.rsp
2# 参考修改参数

登陆oracle用户，进行正式安装：

1[oracle@iZbp15qc4wmx335c268l5mZ ~]$ cd /app/oracle/database
2[oracle@iZbp15qc4wmx335c268l5mZ database]$ ./runInstaller -silent -responseFile /app/oracle/database/response/db_install.rsp
