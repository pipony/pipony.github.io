Java基础之Calendar日历类

Calendar类用于完成日期和时间字段的操作，它可以通过特定的方法设置和读取日期的特定部分，比如年、月、日、时、分和秒等。

定义方法

Calendar类是一个抽象类，不可以被实例化。

因此，需要使用静态方法getInstance()来得到一个Calendar对象

1Calendar calendar = Calendar.getInstance(); // 获取当前日期

获取方法

方法1

可获取字段包括：年、月、日、时、分、秒，需要使用Calendar类中定义的常量值来表示，如Calendar.YEAR用于表示年份。

 1Calendar calendar = Calendar.getInstance();
 2int year = calendar.get(Calendar.YEAR);
 3int month = calendar.get(Calendar.MONTH);
 4int day = calendar.get(Calendar.DATE); // DATE表示日
 5int hour = calendar.get(Calendar.HOUR);
 6int minute = calendar.get(Calendar.MINUTE);
 7int second = calendar.get(Calendar.SECOND);
 8System.out.println(year+"年"+(month+1)+"月"+day+"日"+hour+"时"+minute+"分"+second+"秒");
 9/****执行结果******/
102022年10月25日7时4分37秒

需要特别注意：

在使用Calendar.MONTH字段时，月份的起始值是从0开始的，而不是从1开始，因此要获取当前的月份需要在Calendar.MONTH的基础上加1。

方法2

使用Calendar的getTime()方法，getTime()方法会返回一个表示Calendar时间值的Date对象

1Calendar calendar = Calendar.getInstance();
2System.out.println(calendar.getTime());
3/******执行结果*******/
4Tue Oct 25 07:10:22 CST 2022

设置方法

使用set字段进行设置，有以下三种格式：

方法声明 功能描述
void set(int field,int value) 为指定日历字段设置指定值
void set(int year,int month,int date) 设置Calendar对象的年、月、日三个字段的值
void set(int year.int month,int date,int hourOfDay,int minute,int second) 设置Calendar对象的年、月、日、时、分、秒六个字段的值

格式1:

 1Calendar calendar = Calendar.getInstance();
 2System.out.println("当前日期：");
 3System.out.println(calendar.getTime());
 4// 指定日历字段进行设置
 5calendar.set(Calendar.MONTH, 10); // 注意：实际上设置为10+1=11月
 6calendar.set(Calendar.DATE, 10);
 7System.out.println("设置方式1：");
 8System.out.println(calendar.getTime());
 9/******执行结果*******/
10当前日期：
11Tue Oct 25 07:19:46 CST 2022
12设置方式1：
13Thu Nov 10 07:19:46 CST 2022

格式2：

 1Calendar calendar = Calendar.getInstance();
 2System.out.println("当前日期：");
 3System.out.println(calendar.getTime());
 4// 指定日历字段进行设置
 5calendar.set(2020,8,19); // 设置为2020年9月19日
 6System.out.println("设置方式2：");
 7System.out.println(calendar.getTime());
 8/******执行结果*******/
 9当前日期：
10Tue Oct 25 07:28:16 CST 2022
11设置方式2：
12Sat Sep 19 07:28:16 CST 2020

格式3:

 1Calendar calendar = Calendar.getInstance();
 2System.out.println("当前日期：");
 3System.out.println(calendar.getTime());
 4// 指定日历字段进行设置
 5calendar.set(2020,8,19,20,19,18); // 设置年、月、日、时、分、秒
 6System.out.println("设置方式3：");
 7System.out.println(calendar.getTime());
 8/******执行结果*******/
 9当前日期：
10Tue Oct 25 07:29:58 CST 2022
11设置方式3：
12Sat Sep 19 20:19:18 CST 2020

增减日期方法

使用方法void add(int field,int amount)，amount为正数表示增加，为负数表示减少。

 1Calendar calendar = Calendar.getInstance();
 2System.out.println("当前日期：");
 3System.out.println(calendar.getTime());
 4calendar.add(Calendar.DATE,1); // 日+1
 5calendar.add(Calendar.YEAR,-2); // 年-2
 6System.out.println("日期增减后：");
 7System.out.println(calendar.getTime());
 8/******执行结果*******/
 9当前日期：
10Tue Oct 25 07:33:35 CST 2022
11日期增减后：
12Mon Oct 26 07:33:35 CST 2020

格式化输出日期方法

采用SimpleDateFormat类：

 1SimpleDateFormat myFmt=new SimpleDateFormat("yyyy年MM月dd日 HH时mm分ss秒");
 2SimpleDateFormat myFmt1=new SimpleDateFormat("yy/MM/dd HH:mm");
 3SimpleDateFormat myFmt2=new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");//等价于now.toLocaleString()
 4SimpleDateFormat myFmt3=new SimpleDateFormat("yyyy年MM月dd日 HH时mm分ss秒 E ");
 5SimpleDateFormat myFmt4=new SimpleDateFormat(
 6  "一年中的第 D 天 一年中第w个星期 一月中第W个星期 在一天中k时 z时区");
 7Calendar calendar = Calendar.getInstance();
 8System.out.println(myFmt.format(calendar.getTime()));
 9System.out.println(myFmt1.format(calendar.getTime()));
10System.out.println(myFmt2.format(calendar.getTime()));
11System.out.println(myFmt3.format(calendar.getTime()));
12System.out.println(myFmt4.format(calendar.getTime()));
13/******执行结果*******/
142022年10月25日 07时42分34秒
1522/10/25 07:42
162022-10-25 07:42:34
172022年10月25日 07时42分34秒 周二
