Linux-系统信息查看命令

整理一些关于linux的系统信息查看命令，无逻辑，纯自用。

查看系统版本

1、查看发行版本（适用所有linux系统）

lsb_release -a

2、查看Linux内核版本

uname -a 和 cat /proc/version

 1[cindy@iZbp15qc4wmx335c268l5mZ ~]$ lsb_release -a
 2LSB Version:    :core-4.1-amd64:core-4.1-noarch
 3Distributor ID: CentOS
 4Description:    CentOS Linux release 7.8.2003 (Core)
 5Release:        7.8.2003
 6Codename:       Core
 7[cindy@iZbp15qc4wmx335c268l5mZ ~]$ uname -a
 8Linux iZbp15qc4wmx335c268l5mZ 3.10.0-1127.19.1.el7.x86_64 #1 SMP Tue Aug 25 17:23:54 UTC 2020 x86_64 x86_64 x86_64 GNU/Linux
 9[cindy@iZbp15qc4wmx335c268l5mZ ~]$ cat /proc/version
10Linux version 3.10.0-1127.19.1.el7.x86_64 (mockbuild@kbuilder.bsys.centos.org) (gcc version 4.8.5 20150623 (Red Hat 4.8.5-39) (GCC) ) #1 SMP Tue Aug 25 17:23:54 UTC 2020
